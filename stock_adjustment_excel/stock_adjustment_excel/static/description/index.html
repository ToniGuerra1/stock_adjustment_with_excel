📦 Import Stock Quant from Excel

Actualiza el stock de tus productos directamente desde un archivo Excel (.xlsx), sin complicaciones. Este módulo permite cargar cantidades inventariadas y aplicar los ajustes con solo unos clics.

⚙️ ¿Qué hace este módulo?

  • Exportar productos a Excel con sus cantidades actuales.
  • Cargar un Excel con nuevos valores de stock.
  • Ajustar inventario automáticamente con inventory_quantity y action_apply_inventory.

📂 Estructura del archivo Excel esperada

DEFAULT_CODE   NAME         QUANTITY
P001           Producto A   15.0
P002           Producto B   23.5
  • DEFAULT_CODE: código interno del producto.
  • QUANTITY: cantidad contada del producto.

🔥 Características

  • ✅ Compatible con archivos .xlsx
  • ✅ Interfaz simple e intuitiva
  • ✅ Selección de ubicación (stock.location)
  • ✅ Logs y errores en backend para trazabilidad
  • ✅ Uso de métodos estándar de Odoo

💼 Casos de uso

  • Conteos físicos de inventario
  • Ajustes después de auditorías
  • Importaciones desde Google Sheets o ERPs externos

🧪 Requisitos

  • Librería openpyxl instalada en tu entorno
  • Archivo en formato .xlsx
  • Productos con default_code asignado
